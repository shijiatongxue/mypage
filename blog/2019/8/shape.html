圆形

设置border-radius：

当设置border-radius为一个参数时，这个数字为四个角的半径。由于这里width为100px，所以border-radius应该设置为宽度的一半。

        
          width: 100px;
          height: 100px;
          border-radius: 50px;

椭圆

设置两个参数的时候，分别为长轴长度和短轴长度。注意中间的斜杠必须有。

        
        width: 200px;
        height: 100px;
        border-radius: 100px / 50px;

三角形

        
        width: 0;
        height: 0;
        border-left: 0 solid transparent;
        border-right: 0px solid transparent;
        border-bottom: 100px solid red;

平行四边形

        
        width: 100px;
        height: 100px;
        transform: skewX(-20deg);
        background-color: red;

梯形

        
        width: 100px;
        height: 0;
        border-left: 50px solid transparent;
        border-bottom: 100px solid red;
        border-right: 50px solid transparent;
