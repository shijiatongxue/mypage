HTML5（二）——API

作者：石嘉

日期：2019年8月2日

参考：MDN HTML5

HTML5除了增加了一些标签外，广义上还包括JavaScript和CSS的改变。

通信：Web Sockets，Server-sent events

存储：localStorage，sessionStorage，IndexedDB

多媒体：video，audio

图像：WebGL，SVG

性能&基成：Web Workers，拖放

设备访问：Geolocation

地理位置API通过navigator.geolocation提供。如果该对象存在，说明地理位置服务可用。

样式：box-shadow，border-radius，gradient，transition，transform，text-overflow，multi-columns、flex、grid layouts
