flex的三个参数

flex-grow

定义item的放大比例。它接收一个没有单位的数字作为相应的比例。

如果所有的item设置都为1，则容器的大小被平均分配到各个item；如果一个item设置2，则这个item的大小是设置为1的item的2倍。

        
        .item {
          flex-grow: 0; /* 默认为0 */
        }

这个参数不可以设置为负数。

flex-shrink

定义item的缩小比例，默认大小为1。当空间不足时，item会按比例缩小。如果设置为0，则无论怎样都不缩小。

        
        .item {
          flex-shrink: 1;
        }

负数是不允许的。

flex-basis

它定义元素的默认大小，在剩下的空间被分配之前。它可以设置为长度（20%，5rem等）或者关键词。如果关键词设置为auto，则代表参考它的width和height属性。

        
        .item {
          flex-basis: <length> | auto;
        }

如果flex-basis设置为0，则不考虑content周围的额外空间；如果设置为auto，额外空间会根据flex-grow进行分配。

flex

flex是这三个属性的缩写，其中第二和第三个参数是可选的。flex的默认值为0 1 auto。

        
        .item{
          flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
        }

推荐使用flex写法。

举例：

        
        .wrapper{
          display: flex;
          width: 500px;
          height: 300px;
          background: #333;
          color: #ffffff;
        }
        .left{
          flex: 3 1 50px;
          background: red;
        }
        .right{
          flex: 2 1 100px;
          background: blue;
        }

bind和apply

bind()方法会创建一个新函数。当这个函数被调用时，bind()的第一个参数将作为它运行时的this，之后的一序列参数将会在传递的实参前传入作为它的参数。

        
        var foo = {
          value: 1
        };

        function bar(){
          console.log(this.value);
        }

        var bindFoo = bar.bind(foo);
        bindFoo(); // 1

call和apply则是使用新的this去直接调用、执行函数。

        
        bar.call(foo); // 1
