类数组转化为数组的几种方法

作者：石嘉

日期：2019年7月25日

参考：JavaScript.Info

题目

                
                function fc(a, b, c) {
                    var arr = ?; // 将arguments转化为数组
                    console.log(arr)
                }

原生方法：slice

                    
                    arr = Array.prototype.slice.call(arguments);
                    // 或
                    arr = [].slice.call(arguments);

ES6方法：Array.from

                    
                    arr = Array.from(arguments)

ES6方法：...拓展符号

                    
                    arr = [...arguments];

（完）
