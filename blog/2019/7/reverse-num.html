算法题：输入数组，输出其逆序字符串

作者：石嘉

日期：2019年7月9日

递归实现

                
                    function fc(num){
                        if((''+num).length === 1){
                            return ''+num;
                        }
                        return (''+num).slice(-1) + fc(parseInt((''+num).slice(0,-1), 10));
                    }
                    fc(1234); // "4321"

优化

                
                    function fc(num){
                        if((''+num).length === 1){
                            return ''+num;
                        }
                        return (''+num).slice(-1) + arguments.callee(parseInt((''+num).slice(0,-1), 10));
                    }

其中argument.callee是一个指向正在执行函数的指针，可以用它来实现对函数的递归调用。

优化

在严格模式下，不能通过脚本arguements.callee，访问这个属性会导致错误。不过，可以使用命名函数表达式来达成相同的效果。

                
                    var reverseNum = (function fc(num){
                        if((''+num).length === 1){
                            return ''+num;
                        }
                        return (''+num).slice(-1) + fc(parseInt((''+num).slice(0,-1), 10));
                    });
                    reverseNum(1234); // "4321"
