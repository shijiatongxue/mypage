壹题116：数组 + 数组 = ？

作者：石嘉

日期：2019年7月30日

参考：壹题

题目

        
        console.log([1, 2] + [3, 4]);

分析

对于数组进行二元运算，输出每个元素拼接逗号','。

String()与toString()

二元+运算符在任意一方是字符串时，会把另外一个运算元也转换为字符串。转换的规则是使用String()转换方法。

String()的转换规则是这样的：

  • 如果运算元存在toString()方法，则调用该方法；
  • 对于undefined，转化为"undefined"；
  • 对于null，转换为"null"。

值得注意的是，undefined和null没有toString()方法，所以String()转换方法可以说更广泛一点。

        
          undefined.toString(); // TypeError
          null.toString(); // TypeError

          (123).toString(); // "123"
          true.toString(); // "true"
          "hello".toString(); // "hello"
          Symbol("id").toString(); // "Symbol(id)"

          [1, 2, 3].toString(); // "1,2,3"
          ({}).toString(); // "[object Object]"
          
          let fc = function({console.log("hello")});
          fc.toString(); // "function(){console.log("hello world")}"
