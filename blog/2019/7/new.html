new一个对象

作者：石嘉

日期：2019年7月24日

参考：JavaScript.Info

                
                // 普通的函数被new调用
                function Person(name, age) {
                    this.name = name;
                    this.age = age;
                }
                var p = new Person();

当一个函数被new调用时，会发生下面步骤：

  1. 创建一个空对象，并赋值给this；
  2. 执行函数体。通常会给this添加一些属性；
  3. 返回this的值。
                
                // 等同于上面
                function Person(name, age) {
                    this = {};
    
                    this.name = name;
                    this.age = age
    
                    return this;
                }

（完）
