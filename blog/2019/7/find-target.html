京东快手面试：两数之和

作者：石嘉

日期：2019年7月7日

参考：blockmood

问题

给定一个整数数组和一个目标值，找出数组中和为目标值的两个数。

你可以假设每个输入只对应一种答案，且同样的元素不能被重复利用。

                
                    给定 nums = [2, 7, 11, 15], target = 9

                    因为 nums[0] + nums[1] = 2 + 7 = 9
                    所以返回 [0, 1]

思路

暴力搜索，从第一个元素往后进行匹配，复杂度O(n^2)。

优化

                
                    function fc(arr, target){
                        let map = {};
                        for(let i = 0; i < arr.length; i++){
                            map[arr[i]] = i;
                        }
                        for(let i=0; i < arr.length; i++){
                            let d = target - arr[i];
                            if(map[d]){
                                return [i, map[d]];
                            }
                        }
                        return "Not exists.";
                    }

这个方法采用的是哈希的方法，复杂度为O(n)。
