壹题108：写出如下代码打印结果

作者：石嘉

日期：2019年7月19日

参考：壹题

京东面试题

                
                    var name = 'Tom';
                    (function() {
                        if (typeof name == 'undefined') {
                            var name = 'Jack';
                            console.log('Goodbye ' + name);
                        } else {
                            console.log('Hello ' + name);
                        }
                    })();

打印结果如下：

                
                    Goodbye Jack

在函数内部，只要var命名了变量，无论在哪个部分，都会发生变量提升效果，提升的位置为函数的首行。

                
                    var name = 'Tom';
                    (function() {
                        if (typeof name == 'undefined') {
                            let name = 'Jack';
                            console.log('Goodbye ' + name);
                        } else {
                            console.log('Hello ' + name);
                        }
                    })();

打印结果：

                
                  Hello Tom

let只在块级作用域有效，也不存在变量提升，因此这里的typeof name在函数作用域没有找到name变量后，会去全局作用域查找。查到的是外层var定义的name，因此打印结果为Hello Tom。

（完）
